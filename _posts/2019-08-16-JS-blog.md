JavaScript (usually abbreviated as JS) is an advanced, interpreted programming language. JavaScript is a prototype-based, function-first language, a multi-paradigm language that supports object-oriented programming, imperative programming, and functional programming.Today we learn about if&for


# if语句格式
    if(){}
    判断括号内内容是否为true,为true则执行大括号内的代码,为false则不执行

## if应用

    格式for(定义变量;判断语句;自增){执行语句}

            var i = 1;

			//列出100以内的质数
			var count = 0;  //计数器
			for (var i = 1; i <= 100; i++) { //轮流取出i

					}

            //开方数求质数

             for (var i = 2; i <= 100; i++) { //轮流取出i

             		if (i % j == 0) {
             			//console.log(i)
             			count++;
             		}
             	}
             	if (count == 1) {  //满足条件则输出 此时只能被整除一次，被1整除

## 三目运算
    格式 n = a > b ? g : l;
        如果a大于b则返回g否则返回l
```javascript
<code>
    var a = 10,b = 5;
    var n ;
    n = a > b ? 'yes' : 'no';
    alert(n); //此时n为yes;
</code>
```

## 三目运算筛选最大
```javascript
<code>
    //三目运算进行大小筛选
    var a = 10,b = 20,c = 15;
    var max = a > b ? (a > c ? a : c) : (b > c ? b : c);
	alert(max);
</code>
```
